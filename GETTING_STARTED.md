## ğŸ“– Understanding the Code

### Key Files
- `lib/hub/hub.ts` - Central hub singleton
- `lib/modules/BaseModule.ts` - Module base class
- `modules/*/` - Each module's implementation
- `app/modules/[moduleId]/page.tsx` - Dynamic module routing
- `app/components/modules/*View.tsx` - UI for each module

### Adding a New Module
1. Create `modules/mymodule/MyModule.ts` extending BaseModule
2. Define 4 vertices in the constructor
3. Implement module-specific methods
4. Create `app/components/modules/MyModuleView.tsx` for UI
5. Register in `lib/hub/moduleRegistry.ts`
6. Add routing in `app/modules/[moduleId]/page.tsx`

## ğŸ†˜ Troubleshooting

**Module not loading?**
- Check browser console for errors
- Verify the module is registered in `moduleRegistry.ts`
- Make sure `hub.initialize()` was called

**Styles look wrong?**
- Tailwind uses v4 syntax: `bg-linear-to-r` not `bg-gradient-to-r`
- Check `globals.css` for custom utilities

**TypeScript errors?**
- Restart VS Code's TypeScript server: Cmd+Shift+P > "TypeScript: Restart TS Server"
- Clear node_modules and reinstall: `rm -rf node_modules && npm install`
